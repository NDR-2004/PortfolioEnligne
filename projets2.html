Calculatrice

Ce programme exécute une application graphique, créée avec le framework Avalonia, pour réaliser une calculatrice visuelle simple.

Résumé du projet

Petite calculatrice de bureau permettant d’effectuer des opérations arithmétiques de base.

  • Une fenétre graphique
  • une zone de saisie
  • Des boutons pour écrire les caculs dans la zone de texte
  • Le bouton '=' pour réaliser le calcul et l'afficher dans la zone de texte

Stack technique

  • C#
  • Avalonia
  • Visual Studio Code

Architecture

Le projet est structuré en plusieurs modules :

  • App.axaml - Thème global Avalonia
  • MainWindow.axaml – Interface graphique (boutons + zone de texte)
  • MainWindow.axaml.cs – Logique C# associée à la fenêtre
  • Program.cs – Point d’entrée
  • CalculatriceCsharp.csproj – Projet .NET
  • README.md – Documentation
